# ADR 0001 — Backstage como portal de desenvolvimento

## Decisão

Usar Backstage como catálogo de software, TechDocs e visualização operacional somente leitura da plataforma FCS.

## Consequências

Os descritores permanecem nos repositórios que eles descrevem. O portal lê o GitHub por meio de uma GitHub App com permissões somente leitura. Não há ações de scaffolding ou deploy nesta PoC.
